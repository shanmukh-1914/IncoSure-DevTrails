
  # IncoSure

  This is a code bundle for IncoSure. The original project is available at https://www.figma.com/design/11uUfF1go6sJTDTKZlwZGj/IncoSure.

  ## Running the code

  Frontend setup and run:

  Run `cd frontend`.

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Backend setup and run:

  Run `cd backend`.

  Run `mvn spring-boot:run` to start the backend API.
